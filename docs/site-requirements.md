# 高校スペイン語教師の会 公式ホームページ 要件メモ

更新日: 2026-03-07

## 1. サイト概要
- サイト名: 高校スペイン語教師の会 公式ホームページ
- サイト種別: 新規結成された会の公式サイト
- 想定公開方法: GitHub Pages + Jekyll

## 2. 主な目的
- 会の存在と目的を伝える
- 高校でスペイン語を教える教員同士のつながりを作る
- これからスペイン語教師を目指す人に情報を届ける
- 活動内容やイベント情報を発信する
- 入会や問い合わせの窓口を用意する

## 3. 想定利用者
### 主対象
- 高校でスペイン語を教えている教師
- これからスペイン語教師を目指す人

### 想定来訪者
- 大学の先生
- 別言語を教えている先生
- 教育関係者
- 会の活動に関心のある一般来訪者

## 4. 必要ページ
- トップページ
- 高校スペイン語教師の会とは
- お知らせ
- イベント
  - これからのイベント
  - 過去のイベント
- 入会やお問い合わせについて
- プライバシーポリシー

## 5. 追加を推奨するページ
### 優先度: 高
- トップページ
  - 各ページへの導線をまとめるため必須
- 404ページ
  - GitHub Pages でも用意しておくと親切

### 優先度: 中
- 会の目的・設立趣旨
  - 「高校スペイン語教師の会とは」に含めてもよい
- よくある質問
  - 入会条件、対象者、連絡方法などを整理しやすい
- 運営体制 / 役員紹介
  - 公開できる範囲で信頼性向上に役立つ

### 優先度: 将来検討
- 活動記録
  - 年度ごとの活動アーカイブ
- 資料・リンク集
  - 授業実践、教材、参考リンクなど

## 6. 現時点の構成案
1. トップページ
2. 会について
3. お知らせ一覧
4. イベント一覧
5. 入会・お問い合わせ
6. プライバシーポリシー
7. 404ページ

## 7. 検討事項
- 問い合わせ方法はフォームかメールリンクか
- 入会申込をどこまでオンライン化するか
- 会員限定ページを作る予定があるか
- 更新担当者は誰か
- 写真掲載やイベントレポート時の個人情報配慮をどうするか
- 将来、多言語対応が必要か

## 8. 初期方針メモ
- まずは公開に必要な基本ページを優先する
- GitHub Pages / Jekyll の標準的な構成で始める
- 運用負荷を抑えるため、更新しやすいページ設計にする
- 個人情報を扱う場合はプライバシー面を先に確認する
