# 高校スペイン語教師の会 公式ホームページ サイトマップ案

更新日: 2026-03-07

## 1. 基本方針
- 初期公開では、必要性の高い基本ページを優先する
- ナビゲーションは分かりやすく、項目数を増やしすぎない
- GitHub Pages + Jekyll で運用しやすい構成にする
- お知らせとイベントは継続更新しやすい設計にする

## 2. グローバルナビゲーション案
- ホーム
- 高校スペイン語教師の会とは
- お知らせ
- イベント
- 入会・お問い合わせ

※ プライバシーポリシーはフッター配置を基本とする

## 4. ページ別の目的と内容

## ホーム
### 目的
- 初めて来た人に会の概要を短時間で伝える
- 各主要ページへ迷わず移動できるようにする

### 主な掲載内容
- キャッチコピー
- 会の概要紹介
- 主要導線ボタン
- 最新のお知らせ 3件程度
- 直近イベント 3件程度
- 入会案内への導線

## 高校スペイン語教師の会とは
### 目的
- この会の存在意義と活動方針を説明する

### 主な掲載内容
- 設立の背景
- 会の目的
- 想定する参加者
- どのような活動を行うか
- 今後の展望

## お知らせ
### 目的
- 更新情報を時系列で案内する

### 主な掲載内容
- お知らせ一覧
- 投稿日
- カテゴリ（必要なら）
- 各記事へのリンク

## イベント
### 目的
- 開催予定と過去実績を整理して見せる

### 主な掲載内容
- 開催予定イベント一覧
- 過去イベント一覧
- 開催日
- 会場 / 開催方式
- 申込案内
- 開催レポートへの導線（将来）

## 入会・お問い合わせ
### 目的
- 入会希望者と問い合わせ希望者の行動を支援する

### 主な掲載内容
- 入会対象の説明
- 入会方法
- 問い合わせ方法
- 返信までの目安
- 注意事項

## プライバシーポリシー
### 目的
- 個人情報の取り扱いを明確にする

### 主な掲載内容
- 取得する情報
- 利用目的
- 第三者提供の有無
- 問い合わせ先
- 改定方針

## 404ページ
### 目的
- ページが見つからない場合でも、利用者を迷わせない

### 主な掲載内容
- 簡潔な案内文
- ホームへのリンク
- 主要ページへの導線
